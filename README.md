this project relates to "TJ Weather Station" and "TJ Fan Controller"(not published yet) and "TJ Weather Node"(not published yet)

Parts list:
- Seeed Studio XIAO ESP32 S3
- 3 Buttons
- 1pcs 2 way switch
- 3 LED
- 2.42 OLED Screen
- Velleman G1068 box
- 72 X 100 X 2 mm aluminium

After starting to build the weather station I realised that that might be a bit too big to start as my first project so I built the weather node and after that (this is around the time where summer started to get hot) I needed a fan for my room, so I built the fan controller, so then could controll the fan from my phone(HA app) and make automations based on data from the weather node, and I wanted to controll it mechanically (button presses) so I got the idea to build this project. I also didnt want this to be a dead end so I added a mode switch so I can switch between "FAN" and "CUSTOM" modes, so in the future I can controll other devices too. I also wanted to have a screen so I bought a 2.42 OLED, that was the biggest single color I could find. after that I assembled the box using the visio box drawing, and the aluminium
